> **项目名称：企业级聚合支付与资金结算中台**
> 
> **项目描述：** 针对业务线多条异构支付渠道的接入痛点，主导建设统一的聚合支付网关平台。屏蔽底层第三方 API 差异，支撑日均 **10万+** 笔订单的高效流转，确保核心资金链路的高可用与全年 **0 资损**。
> 
> **核心职责与技术亮点：**
> 
> - **高可用异构路由架构：** 采用**策略模式**将不同支付渠道的 API 接入“插件化”，制定标准化的统一支付规约。将新支付渠道的研发接入效率提升 **50% 以上**，实现核心交易代码的强解耦。
>     
> - **零资损的最终一致性保障：** 针对第三方 API 调用超时、异步回调丢失（掉单）等高危场景，废弃传统的粗暴回滚策略。设计并落地基于**“本地状态机挂起 + 延迟 MQ 主动轮询”**的最终一致性补偿机制。有效拦截异常网络抖动，使平台掉单率降至 **0.01% 以下**，彻底解决单边账引发的客诉问题。
>     
> - **网关级熔断与防雪崩隔离：** 引入**舱壁模式**理念，针对不同的第三方支付 API 配置独立的**线程池隔离**。配合超时阻断机制，彻底杜绝因单一外部渠道宕机而拖垮整个平台网关的雪崩效应，保障核心链路 TP99 响应稳定。
